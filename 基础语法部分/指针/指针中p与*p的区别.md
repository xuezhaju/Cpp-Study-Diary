在C++中，`p` 和 `*p` 的区别很重要：

## 1. **`p` - 指针变量本身**
```cpp
int *p;  // p 是指针变量
int a = 10;
p = &a;  // 把a的地址赋给p，这里用的是p（指针变量）

int *q = p;  // 把p的值（地址）赋给另一个指针q
```

## 2. **`*p` - 指针指向的值（解引用）**
```cpp
int *p;
int a = 10;
p = &a;  // p存储a的地址

// 使用*p来访问或修改a的值
*p = 20;    // 现在 a = 20
int b = *p; // b = 20
```

## 具体使用场景：

### 示例1：基本用法
```cpp
int main() {
    int value = 100;
    int *p = &value;  // p存储value的地址
    
    cout << "p (地址): " << p << endl;      // 输出地址
    cout << "*p (值): " << *p << endl;     // 输出100
    cout << "&value: " << &value << endl;  // 和p的值相同
    
    *p = 200;  // 通过指针修改value的值
    cout << "value: " << value << endl;  // 输出200
    
    return 0;
}
```

### 示例2：函数参数传递
```cpp
// 需要修改原始变量的值
void increment(int *p) {
    *p = *p + 1;  // 这里用*p访问值
}

int main() {
    int x = 5;
    increment(&x);  // 传递x的地址
    cout << x << endl;  // 输出6
    return 0;
}
```

### 示例3：数组和指针
```cpp
int arr[] = {1, 2, 3, 4, 5};
int *p = arr;  // p指向数组第一个元素

// p++ 移动指针
// *p 访问当前指向的元素
for(int i = 0; i < 5; i++) {
    cout << *p << " ";  // 输出值
    p++;  // 移动到下一个地址
}
```

## 快速记忆表：

| 表达式 | 含义 | 使用场景 |
|--------|------|----------|
| `p` | 指针变量本身，存储地址 | 赋值、传递、比较地址 |
| `*p` | 指针指向的值 | 读取、修改目标变量的值 |
| `&p` | 指针变量自己的地址 | 很少使用 |
| `p++` | 移动到下一个内存位置 | 数组遍历 |
| `*p++` | 先取值，然后指针后移 | 常见于数组遍历 |

## 常见错误：
```cpp
int *p;
*p = 10;  // 错误！p还没有指向有效的内存

// 正确做法：
int x;
int *p = &x;  // 先让p指向有效的地址
*p = 10;      // 然后才能赋值
```

**简单总结：**
- 当你想操作**地址**（赋值、传递、比较）时，用 `p`
- 当你想操作**值**（读取、修改）时，用 `*p`
