# 洛谷刷题

主要学到了个string()

- 用法：
  1. string(num, char)    可以将char数据类型的输出num次（其他类型的我不知道，反正string类型是不行的好像，我试过了但是报错了）
  2. 从另一个 string 创建
        string original = "Hello World";

        // 拷贝整个字符串
        string copy1(original);         // "Hello World"

        // 从指定位置开始
        string part1(original, 6);      // "World" (从索引6开始)

        // 从指定位置开始，指定长度
        string part2(original, 0, 5);   // "Hello" (从索引0开始，取5个字符)
