# 11.30学习c++日记

### 数组

- 语法：
    1. 数据类型 数据名[ 数据长度 ]；
    2. 数据类型 数据名[ 数据长度 ] = { 值1, 值2...... }；
    3. 数据类型 数据名[  ] = { 值1, 值2...... }；
   
- 特点：
    1. 要放到一块连续的内存空间中
    2. 数组中的每个元素都是相同的数据类型
    3. 索引数组从0开始 如：arr[0]  #这个叫下标，我们可以通过下标来访问数组中的元素


### 数组名称的作用
1. 可以统计整个数组在内存中的长度     sizeof(arr)   或者可以统计出来每一个元素的大小  sizeof(arr[0])     用 sizeof(arr)//sizeof(arr[0]) 可以获得有多少个元素
2. 可以获得数组在内存中的首地址
