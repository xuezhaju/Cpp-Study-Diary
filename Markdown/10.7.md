# 主要学习了 逻辑运算符 与 if 与 三目运算符 与 swich语法

## 逻辑运算符
- ！  ：   非
- ||  ：  或
- &&  ：  与

## if 
- 就那样懒得写



## 三目运算符
- 语法： 表达式1 ? 表达式2 : 表达式3
- 解析：如果表达式1为真，执行表达式2;否则执行表达式3
- 三目运算符返回的是 **变量** ，可以再进行赋值操作

## swich语法
    swich(表达式){    //swich中只能放 整数 或者 字符型（缺点）   优点：结构清晰，执行效率高

        case 结果1: 执行语句; break;   //break表示退出当前的分支

        case 结果1: 执行语句; break; 

        ...

        default: 执行语句; break;  //默认执行的

    }
