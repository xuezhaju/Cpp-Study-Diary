## 循环
- 语法：for(起始表达式；条件表达式；末尾循环体){循环语句；}
- 示例代码
    int main()
    {

        for(int i = 0; i < 10; i++)
        {
            cout << i << endl ;
        }

        return 0 ;

    }

- 解析： 
    1. 起始表达式可以用与创建变量
    2. 条件表达式为循环条件
    3. 末尾表达式是在 **每次** 循环结束之后进行的语句
