# 乱七八糟笔记

## 2026.06.13
- sizeof()在编译的环节就被求值了，并非函数
- 直接声明一个变量， 例如int a;,不给予一个特殊值，那么这个变量所对应的值是一个不确定的随机数
- 字符类型就整数，在内存中的表达就是整数
- 字符的字面表达只支持ASCII字符，特殊字符需要转义
- char输出将ASCII中的对应的掩码转换成相对应的字符
- 在cout输出的内容中的变量前面加上括号，并在括号中填入想要强制转换的数据类型即可进行强制转换
- chat类型只有一个字节的内存大小,一个字节是8位 
- cpp中不存在 elseif 语法,我们所写的else if 实际上是对 else //换行  if{}嵌套的缩写
- 减少嵌套层数可以提高运行效率
