# 内存分区模型

- C++程序在执行程序的时候，将内存大方向分为 *4个区域* 
1. 代码区：存放函数体的二进制代码，由操作系统管理的
2. 全局区：存放全局变变量和静态变量以及常量
3. 

## 代码区
- 存放CPU执行的机器指令
- 代码区是 **共享** 的，共享的目的是对于频繁执行的程序，在内存中只要有一份代码即可
- 代码区是 **只读** 的，只读的目的是防止程序意外的修改了它的指令

## 全局区
- 全局变量以及静态变量存放于此(Tips:静态变量的前缀是*static*)
- 全局区还包括了常量区，字符串常量以及其他常量也存放于此
    - Tips: 如果前面加上了const修饰，还是一样的，局部变量该在哪在哪，该在全局区的还是在全局区
- 该区域的数据在程序结束后有系统自动释放
- ***总结就是不在全局区里面的有 局部变量、const修饰的局部变量、局部常量***
- ***全局区里面的有 全局变量、静态变量、static关键词修饰的、常量（const修饰的全局变量，字符串常量）***
- C++程序运行之前分为全局区和代码去区
