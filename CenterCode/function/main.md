# 函数提高

## 1.函数的默认参数
- Tip:在函数中如果给了一个函数的形参初始值，那么这个形参后面的形参都要有初始值
```cpp
int func(int a, int b = 10, int v = 20)
{
    return 0;
}

```

- 如果函数的声明有默认参数，那么函数的实现就不能有默认参数,否则会出现 二义性 的问题
```cpp
 int function(int m = 10, int n = 20);

 int function(int m, int n)
 {
    return m + n;
 }

```

- 占位参数
```cpp
void func1(int a, int = 10)
{
    // 占位参数只要写一个数据类型即可, 同时占位参数也可以有一个默认值
    cout << a << endl;
}

```

- 函数重载
- 需要满足的条件：
1. 在同一个作用域下
2. 函数名称相同
3. 函数参数的类型不同 或者 个数不同 或者顺序不同
4. Tips:函数的返回值不可以作为函数重载的条件
-作用：让函数名相同，提高函数的复用性
