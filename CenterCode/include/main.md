# 引用

## 引用的基本语法
- 作用：给变量取别名
- 语法：数据类型 &别名 = 原名 
- 本质：给变量起别名

## 引用的注意事项
- 引用必须初始化
- 引用在初始化之后，不可以改变

```cpp
int main()
{
    int a = 10;
    int &b = a;    
    // int &b ;错误，引用必须初始化

    int &c = a;
    c = b;  //这是赋值操作，没用改变引用

    return 0;
}
```

## 引用做函数参数
- 作用：函数传参的时候，可以利用引用的技术让形参修饰实参
- 优点：可以简化指针修改实参(见cpp文件 **funcSwap.cpp**)

## 引用做函数的返回值
- 作用：引用可以作为函数的返回值存在的
- 注意：**不要返回局部变量的引用**
- 用法：函数调用作为左值
