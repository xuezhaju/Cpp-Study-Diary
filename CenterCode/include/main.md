# 引用

## 引用的基本语法
- 作用：给变量取别名
- 语法：数据类型 &别名 = 原名 
- 本质：给变量起别名

## 引用的注意事项
- 引用必须初始化
- 引用在初始化之后，不可以改变

```cpp
int main()
{
    int a = 10;
    int &b = a;    
    // int &b ;错误，引用必须初始化

    int &c = a;
    c = b;  //这是赋值操作，没用改变引用

    return 0;
}
```

## 引用做函数参数
- 作用：函数传参的时候，可以利用引用的技术让形参修饰实参
- 优点：可以简化指针修改实参(见cpp文件 **funcSwap.cpp**)

## 引用做函数的返回值
- 作用：引用可以作为函数的返回值存在的
- 注意：**不要返回局部变量的引用**
- 用法：函数调用作为左值

## 引用的本质
- 源代码：int& a = b;
- 编译器自动转换过后的代码：int* const a = &b;  (是一个指针常量,指向的内存不变，值可以改变)
- 同样的如果编译器发现是一个引用的情况下会这样
- a = 20;   ====>   实质上是  *a = 20;  是一个解引用
- 这个例子不太恰当，指针常量需要占用内存空间，但是引用实际上不需要，引用的底层原理应该是编译时候生成的符号表，引用名称和原名称均是一个符号，一个符对应一个地址，两符号对应相同地址，后续链接直接换成地址

## 常量引用
- 作用：常量引用主要是用来修饰形参，防止误操作
- 在函数列表中，可以加const修饰形参，防止形参修改实参
```cpp
int main()
{
    const int & ref = 10 ;   // 编译器会自动创建一个临时变量，  int temp = 10; int& ref = temp;  找不到原名的
    // ref = 20 ; 是不允许的，因为加上了const修饰，为只读不可修改
}



```
